# Elementum Landing Page Assignment

A clean, responsive landing page built with React, Vite, and Tailwind CSS based on the provided Figma designs.

## 📁 Project Structure
- All active React components, styling configurations, and layout architecture are located inside the `src/` folder.
- Modular sections can be found under `src/components/`.

## 🛠️ Tech Stack
- React
- Vite
- Tailwind CSS
- Lucide React (Icons)
